Sistema de Catraca

O sistema deve:
1.	Reconhecer rostos autorizados
2.	Acender luz em caso de ambiente escuro
3.	Ativar atuador da catraca caso rosto reconhecido
4.	Bloquear a entrada caso não reconhecido e sinalizar
5.	Armazenar funcionário, data e hora de entrada e saída 
6.	Armazenar rostos em banco de dados separado e seguro
7.	Exibir os dados adequados corretamente e com opções de filtragem
8.	Permitir que o usuário veja os dados brutos sem formatação
